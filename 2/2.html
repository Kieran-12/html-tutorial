Images:

Images really spice up your website, providing visual detail to what your website is trying to do. But how do you insert a picture?
You can insert a picture by using the <img> tag. When nothing is inside it, it will look like this:

<img src="" alt="" width="" height="">

By default, the 'width' and 'height' parts won't be there, although they are still important to define the size of the image on the website. The computer reads width="100" height="150" as the width=100 pixels and height=150 pixels and the height is 150 pixels. Inside 'alt', short for 'alternative text', you can write text that replaces the image in case the image cannot load. It is useful to write text to describe the image, which will provide the reader with the necessary information that the image would have otherwise provided. Inside 'src' short for 'source' is where the image can be found. Usually, you have to move around inside the folder to find the image. For example:

<img src="../images/youtube_logo.jpg" alt="YouTube Logo">

The '..' part is important as it tells the computer to exit the current location and start searching from there. Your folders may be arranged in a different way so your code may look a little different. Just make sure you remember that you always start from wherever the code is actually written. An easier way to understand this is let's say this html code is in a folder called 'html' and the image is in a seperate folder called 'images'. To get to this picture, we must exit 'html' through '..' then enter images through '/images' then access the image through '/youtube_logo.jpg'. This is what the code looks like:

Links:

To put a link on the website, you can use the <a> tag. For example, if we want the link to send the user to the website 'https://www.youtube.com/', we can write the code in the 'href' part. Below is the code.

<a href="https://www.youtube.com/">YouTube Link</a>

The link below opens a new YouTube tab:

YouTube Link

These are called hyperlinks. You may have seen a lot of these on Wikipedia. When clicked, they send you to a different website.

You can also use an image as a link. It's the same as a normal text hyperlink, where when you click on the picture, it will send you to a different website. For example:

<a href="https://www.youtube.com/"><img src="youtube_logo.jpg" alt="YouTube Logo" width="180" height="120"></a>
